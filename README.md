Main Modules of the LMS
1. Admin Panel

Admin can:

Manage Users (Students/Teachers)

Add/Update/Delete Courses

Manage Videos, Files, Assignments

Manage Payments & Orders

View Reports (sales, enrollments, quiz results)

Control system settings

2. Teacher Panel

Teachers can:

Create new courses

Upload video lessons (YouTube/Local file)

Upload PDFs, PPTs, files

Create quizzes & assignments

Check student submissions

Post announcements

3. Student Panel

Students can:

Create an account

Browse free & paid courses

Buy premium courses

Watch lessons after purchase

Download materials

Take quizzes

Submit assignments
